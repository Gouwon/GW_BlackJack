HTML <b> and <strong> Elements

HTML <b> 요소는 추가적인 중요성이 없는 볼드 텍스트를 정의한다.
HTML <strong> 요소는 의미상 "강한" 중요성을 추가하는 강한 텍스트를 정의한다.

<b>This text is bold</b>
<storng>This text is strong</strong>
HTML <i> and <em> Element

HTML <i> dytsms 추가적인 중요성이 없는 이태릭체를 정의한다.
HTML <em> 요소는 의미상 중요성을 추가하는 강조된 텍스트를 의미한다.

<i>This text is italic</i>
<em>This text is emphasized</em>
참고: 브라우저는 <strong>를 <b>로, <em>를 <i>로 나타낸다. 그러나, 이 태그들 사이에는 의미상의 차이가 있다. <b>와 <i>는 볼드체와 이태릭체를 나타내지만, <strong>와 <em>는 그 텍스트가 '중요'하다는 것을 의미한다.

HTML Small Formatting

HTML <mark> Elemenent

HTML <mark> 요소는 표신되거나 강조된 텍스트를 의미한다.

HTML Marked Formatting

<h2>HTML <mark>Marked</mark> Formatting</h2>
HTML <del> Element

HTML <del> 요소는 삭제되거나 제거된 텍스를 정읳나다.

My favorite color is blue red.

<p>My favorite color is <del>blue</del> red.</p>
HTML <ins> Element

HTML <ins> 요소는 삽입되거나 추가된 텍스트를 의미한다.

My favorite color is red.

<p>My favorite <ins>color</ins> is red.</p>
HTML <sub> Element

HTML <sub> 요소는 아랫첨자가 된 텍스트를 의미한다.

This is subscripted text.

<p>This is <sub>subscripted</sub> text.</p>
HTML <sup> Element

HTML <sup> 요소는 윗첨자된 텍스를 의미한다.

This is superscripted text.

<p>This is <sup>superscripted</sup> text.</p>
