HTML Paragraphs

<p> 요소는 문단을 정의한다.

HTML Line Breaks

HTML <br> 요소는 개행을 의미한다.
만약 새로운 문단에서 시작하지 않고, 줄을 끝내고 새로운 줄에서 시작하기 원한다면, <br>를 사용한다.
<br> 태그는 빈 태그임으로 닫는 태그가 없다.

This is
a paragraph
with line breaks.

<p>This is<br>a paragraph<br>with line breaks.</p>
The HTML <pre> Element

HTML <pre> 요소는 포멧된기전의 텍스트를 정의한다.
<pre> 요소 안의 텍스트는 보통 Courier와 같은 고정폭 폰트로 나타나며, 공백과 개행을 그대로 저장한다.
