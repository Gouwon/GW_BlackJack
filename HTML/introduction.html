Web Browser

웹 브라우저의 목적은 HTML 문서를 읽고, 이를 나타내는 것이다.
브라우저는 HTML 태그들을 나타내지는 않지만, 어떻게 문서를 나타낼지 결정하는 데 이를 사용한다.

HTML Page Structure
주의: <body> 안에 있는 것만 브라우저에 나타난다.
<!DOCTYPE> Declaration

<!DOCTYPE> 선언은 문서 형식을 나타내며, 브라우저가 올바르게 웹 페이지를 나타내는 데 도움을 준다.
선언은 어떤 HTML 태그들보다도 제일 먼저 문서의 최상단에 한번만 나타나야 한다.
<!DOCTYPE> 선언은 대소문자를 가리지 않는다.
HTML5에 대한 <!DOCTYPE> 선언은 <!DOCTYPE html> 이다.
