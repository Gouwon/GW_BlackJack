Styling HTML with CSS

CSS는 Cascading Style Sheets의 약자이다.
CSS는 HTML 요소가 화면이나 문서, 혹은 다른 미디어에서 어떻게 그려질 것인지를 표현한다.
CSS는 많은 작업을 줄여준다. 한 번에 여러 웹 페이지의 레이아웃을 조작할 수 있다.
CSSsms 3가지 방법으로 HTML 요소에 추가될 수 있다.

  • Inline - HTML 요소 안의 스타일 속성을 사용
  • Internal - <head> 태그 내의 <style> 요소를 사용
  • External - 외부 CSS 파일을 사용
CSS를 추가하는 가장 일반적인 바법은 별개의 CSS 파일의 스타일들을 저장하는 것이다.
