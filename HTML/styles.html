Background Color

CSS background-color 속성은 HTML 요소의 배경색을 정의한다.

<body style="background-color:powderblue;">
Text Color

CSS color 속성은 HTML 요소의 텍스트 색을 정의한다.

<h1 style="color:blue;">This is a heading</h1> <p style="color:red;">This is a paragraph.</p>
Fonts

CSS font-family 속성은 HTML 요소에 대한 사용될 폰트 정의한다.

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>
Text Size

CSS font-size 속성은 HTML 요소에 대한 텍스트 크기를 정의한다.

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>
Text Alignment

CSS text-align 속성은 HTML 요소의 텍스트의 수평적 위치를 정의한다.

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>
Chapter Summary
  • style 속성은 HTML 요소들의 스타일링을 할 때 사용한다.
  • background-color 속성은 배경색 설정을 한다.
  • color 속성은 텍스트 컬러를 설정한다.
  • font-family 속성은 텍스트 폰트를 설정한다.
  • font-size 속성은 텍스트의 크기를 설정한다.
  • text-align 속성은 텍스트 위치를 조정할 때 사용한다.
