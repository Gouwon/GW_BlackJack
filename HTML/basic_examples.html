HTML Documents

모든 HTML 문서들은 문서 형식 선언인 <!DOCTYPE html> 과 시작해야 한다.
HTML 문서 자체는 <html>로 시작하여 </html>로 끝나야 한다.
HTML 문서의 보이는 부분은 <body>와 </body>의 사이이다.

HTML Headings

HTML 제목은 <h1>부터 <h6> 태그들 사이이다.
<h1>는 가장 중요한 제목을 의미한다. <h6>는 가장 덜 중요한 제목을 의미한다.

HTML Paragraphs

HTML 문단은 <p> 태그로 나타낸다.

HTML Links

HTML 링크는 <a>태그로 정의된다.
링크의 목적지는 href속성 안에서 구체화된다.
속성들은 HTML 요소들에 추가적인 정보를 제공하기 위해서 사용된다.

HTML Images

HTML 이미지들은 <img> 태그로 정의된다.
소스 파일 주소(src)와 대체 텍스트(alt), width, height; 등이 있다.

HTML Buttons

HTML 버튼들은 button 태그로 정의된다.

HTML Lists

HTML 리스트들은 <ul>(비정렬 리스트), 혹은 <ol>(정렬 리스트) 정의되며, <li> 태그(리스트 항목)들이 따라온다.
