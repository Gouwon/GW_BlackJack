HTML Links - Hyperlinks

HTML 링크들은 하피어링크들이다.
링크를 클릭하면 다른 문서로 점프한다.
링크 위로 마우스를 가져다 놓으면, 마우스 화살표는 작은 손으로 변한다.

HTML Links - Syntax

하이퍼링크는 HTML <a> 태그로 정의한다.
<href> 속성은 목적지 주소를 정의한다.
link text 부분은 보이는 부분이다.
링크 텍스트를 클릭하면 지정된 주소로 보낸다.

link text 
<a href="url">link text</a>
참고: 부폴더 주소의 끝에 '/'가 없다면, 서버에 2개의 요청을 보내게 될 지 모른다. 많은 서버들은 자동적으로 주소의 끝에 '/'를 추가하고, 그리고 나서 새로운 요청을 만든다.
Local Links

주소에는 절대주소가 있다.
로컬 링크는 상대 주소이다.

HTML Link Colors

기본적으로 링크는 모든 브라우저들에서 다음과 같이 나타난다.

  • 아직 방문하지 않은 링크는 밑줄과 파란색이다.
  • 방문한 링크는 밑줄과 보라색이다.
  • 활성화된 링크는 밑줄과 빨간색이다.
CSS를 이용하여 기본값을 바꿀 수 있다.

<style>
    a:link {
        color: green; 
        background-color: transparent; 
        text-decoration: none;
    }
    
    a:visited {
        color: pink;
        background-color: transparent;
        text-decoration: none;
    }
    
    a:hover {
        color: red;
        background-color: transparent;
        text-decoration: underline;
    }
    
    a:active {
        color: yellow;
        background-color: transparent;
        text-decoration: underline;
    }
</style>
HTML Links - The target Attribute

target 속성은 연결된 문서를 어디서 열 것인지를 정의한다.
target 속성은 다음 값들 중 하나를 가질 수 있다.

  • _blank - 새로운 창이나 탭에서 링크된 문서를 연다.
  • _self - 링크를 클릭하면 같은 창이나 탭에서 링크된 문서를 연다(기본).
  • _parent - 링크된 문서를 부모 프레임에서 연다.
  • _top - 링크된 문서를 창의 전체 본문에서 연다.
  • framename - 링크된 문서를 해당 프레임에서 연다.

팁: 만약 웹페이지가 프레임 안에 있다면, target="_top"으로 해당 프레임을 벗어날 수 있다.
HTML Links - Image as Link

이미지를 링크로 사용할 수 있다.

<a href="default.asp">
    <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;border:0;">
</a>
참고: border:0;가 추가되면 IE9(이전 버전)에서 이미지 주변에 경계선이 나타나는 것을 예방할 수 있다.

HTML Links - Create a Bookmark

HTML 북마크는 사용자들을 웹 페이지의 특정한 부분으로 이동할 수 있게끔 할 때 사용된다.
북마크는 웹페이지가 길 때, 유용할 수 있다.
북마크를 만들기 위해서는 먼저 북마크를 생성하고, 거기에 링크를 추가해야 한다.
