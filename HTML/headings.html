HTML Headings

제목들은 <h1>에서 <h6> 태그 사이로 정의된다.
<h1> 태그는 가장 중요한 제목을 정의한다. <h6> 태그는 가장 덜 중요한 제목을 정의한다.

참고: 브라우저는 자동적으로 제목 앞뒤로 margin을 추가한다.
Headings Are Important

검색엔진들은 웹페이지의 내용물과 구조를 인덱스하는 데 제목들을 사용한다.
사용자들은 종종 페이지를 제목을 통해서 훑어본다. 문서 구조를 보여주는 데 제목들을 사용하는 것은 중요하다.
<h1> wpahrdms 중심 제목으로 사용되어야 하며, 그 다음으로는 <h2> 제목, 그리고 덜 중요한 <h3>의 순으로 진행한다.

참고: HTML 헤딩들은 제목들로만 사용한다. 헤딩들을 크거나 두껍개 만들지 말아야 한다.
Bigger Headings

각각의 HTML 헤딩들은 기본 크기가 있다. 그러나 CSS의 font-size 속성을 사용하여 style과 함께 제목에 크기를 나타낼 수 있다.

<h1 style="font-size:60px;">Heading 1</h1>
HTML Horizontal Rules

<hr> 태그는 HTML 페이지 안에서 의미적 구분을 정의하고, 수평적 규칙으로서 가장 많이 보여진다.
<hr> 요소는 HTMl 페이지에서 내용물을 분리하거나 변화를 정의할 때 사용된다.

The HTML <head> Element

HTML의 <head> 요소는 메타데이터(metadata)를 위한 컨테이너이다. HTML 메타데이터는 HTML 문서에 대한 데이터이다. 메타데이터는 그려지지 않는다.
<head> 요소는 <html> 태그와 <body> 태그 사이에 위치한다.

참고: 메타데이터 일반적으로 문서 제목, 문자 설정, 스타일, 스크립트 그리고 다른 메타 정보를 정의한다.
HTML Tag Reference
Tag Description
<html> HTML 문서의 최상위를 정의한다.
<body> 문서의 본문을 정의한다.
<head> 제목, 스크립트, 스타일, 메타 정보 등 모든 head 요소들의 컨테이너이다.
<h1> to <h6> HTML 제목
<hr> 내용물 안에서 의미적 변화를 정의한다.
